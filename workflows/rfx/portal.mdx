## Overview

Many organizations use online portals for RFx submissions. Tribble's browser extension helps streamline the process of responding to these portal-based questionnaires and forms.

## Supported Platforms

- Ariba
- SAP SRM
- Coupa
- RFP360
- Custom vendor portals
- General web forms

## Getting Started

### 1. Browser Extension Setup

1. **Installation**
   - Install Tribble browser extension
   - Sign in to your Tribble account
   - Pin extension for easy access

2. **Project Creation**
   - Create new RFx project in Tribble
   - Add portal URL to project
   - Set up project parameters

### 2. Portal Integration

1. **Connection Setup**
   - Navigate to vendor portal
   - Open Tribble extension
   - Link portal to project

2. **Form Recognition**
   - Extension scans form fields
   - Identifies question types
   - Maps to response categories

## Troubleshooting

### Common Portal Issues

1. **Connection Problems**
   - Check internet connection
   - Verify portal access
   - Update browser extension

2. **Form Recognition**
   - Refresh page
   - Clear browser cache
   - Try alternative browser

3. **Auto-fill Issues**
   - Check field compatibility
   - Verify response format
   - Review character limits

### Portal-Specific Tips

#### Ariba
- Use single-session mode

#### RFP360
