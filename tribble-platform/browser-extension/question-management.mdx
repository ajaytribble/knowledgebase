## Question Organization

- Select text directly from portal
- Use keyboard shortcuts for efficiency
- Capture single or multiple questions
- Maintain question context

### Question Categories

- Section/Category
- Priority
- Status
- Assigned SME

## Response Management

### Generated Responses
<Steps>
  <Step title="Review">
    Check response accuracy and relevance
  </Step>
  <Step title="Edit">
    Customize for specific context
  </Step>
  <Step title="Format">
    Match portal requirements
  </Step>
  <Step title="Submit">
    Copy response back to portal
  </Step>
</Steps>

### Response Options
<AccordionGroup>
  <Accordion title="Answer Length">
    - Default (~100 words)
    - Shorter (~50 words)
    - Longer (~150 words)
  </Accordion>
  <Accordion title="Response Style">
    - Technical detail level
    - Formal vs. conversational
    - Include/exclude examples
  </Accordion>
</AccordionGroup>

## Best Practices

### Organization Tips
<Check>Group related questions together</Check>
<Check>Use consistent naming conventions</Check>
<Check>Apply relevant tags</Check>
<Check>Track dependencies between questions</Check>

### Quality Control
<Warning>
Before marking complete:
- Verify response accuracy
- Check character limits
- Confirm formatting
- Review attachments if needed
</Warning>

| Action | Shortcut |
|--------|----------|
| Tag Question | Ctrl/Cmd + T |
| Add Comment | Ctrl/Cmd + M |
| Mark Complete | Ctrl/Cmd + D |
| Request Review | Ctrl/Cmd + R |
