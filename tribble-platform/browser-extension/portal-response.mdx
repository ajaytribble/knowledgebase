The Tribble browser extension transforms how teams handle RFx questionnaires by bringing AI-powered assistance directly to vendor portals. This seamless integration enables teams to leverage their entire knowledge base while working in external systems, dramatically reducing response time and improving answer quality.

Key benefits include:
- AI-generated responses based on your company's knowledge base
- Consistent messaging across all submissions
- Progress tracking and completion monitoring
- Real-time collaboration with team members

## Portal Response Process

<Steps>
  <Step title="Create Project">
    Click the Tribble extension icon and select "New Project" to begin
  </Step>
  <Step title="Select Questions">
    Highlight questions in the portal to capture them in Tribble
  </Step>
  <Step title="Generate Responses">
    Let Tribble generate appropriate responses based on your knowledge base
  </Step>
  <Step title="Review & Submit">
    Review, edit if needed, and paste responses back into the portal
  </Step>
</Steps>

## Working with Questions

### Capturing Questions
1. Highlight the question text in the portal
2. Click the Tribble extension icon
3. Select "Add Question" or use keyboard shortcut (Ctrl/Cmd + Shift + Q)

<Tip>
You can capture multiple questions at once by highlighting entire sections
</Tip>

### Managing Responses
- Review generated responses
- Edit for context or specificity
- Format text as needed
- Track completion status

## Keyboard Shortcuts

| Action | Windows/Linux | macOS |
|--------|--------------|-------|
| Add Question | Ctrl + Shift + Q | Cmd + Shift + Q |
| Generate Response | Ctrl + Shift + G | Cmd + Shift + G |
| Save Progress | Ctrl + Shift + S | Cmd + Shift + S |
| Next Question | Ctrl + Shift + N | Cmd + Shift + N |
