## Dashboard Elements

### Product Usage Statistics

<Frame>
  <img src="/images/web-app/dashboard1.png" alt="Product Usage Statistics" />
</Frame>

* Overall usage metrics for your organization
* Breakdowns by user and feature (e.g., browser extension vs. agent interactions)

### RFx and Questionnaire Project Dashboard

<Frame>
  <img src="/images/web-app/dashboard2.png" alt="RFx and Questionnaire Project Dashboard" />
</Frame>

* Overview of current projects
* Project status, owners, deadlines, and Tribble confidence score

Toggle Dashboard views using the drop-down in the top left, underneath the welcome message

### Time Period Filtering

* Options to view data for different time ranges:
   * This month
   * Last 30 days
   * Last 90 days
   * Past year
