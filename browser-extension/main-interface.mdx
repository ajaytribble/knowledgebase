## Overview

The Tribble browser extension interface is designed for simplicity and efficiency. Here's a breakdown of the main components:

### Toolbar Icon

The Tribble icon in your browser toolbar provides quick access to:
- Current project status
- Recent questions
- Quick actions menu

### Main Panel

When you click the Tribble icon, the main panel opens with these sections:

1. **Project Selector** - Choose or create projects
2. **Question List** - View and manage your questions
3. **Settings** - Access your preferences and configuration

### Context Menu

Right-click anywhere on a webpage to access Tribble's context menu for:
- Adding questions
- Creating snippets
- Saving references
