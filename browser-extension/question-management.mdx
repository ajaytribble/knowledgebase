## Creating Questions

### Quick Add

1. Highlight text on any webpage
2. Right-click and select "Add to Tribble"
3. Choose or create a project
4. Add any additional context
5. Click "Save"

### Manual Entry

1. Click the Tribble icon
2. Select "New Question"
3. Enter your question details:
   - Question text
   - Context
   - References
   - Tags

## Managing Questions

### Organization

- Group related questions
- Add tags for easy filtering
- Set priority levels
- Add custom metadata

### Review and Editing

- Edit question content
- Update references
- Manage responses
- Archive or delete questions
