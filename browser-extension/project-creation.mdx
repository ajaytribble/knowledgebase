## Creating a New Project

### From the Extension

1. Click the Tribble icon in your toolbar
2. Select "New Project" from the project selector
3. Enter your project details:
   - Project name
   - Description (optional)
   - Team members (optional)
4. Click "Create Project"

### Project Templates

You can start with a template to speed up project creation:
- Blank Project
- Research Project
- Documentation Project
- Team Collaboration
