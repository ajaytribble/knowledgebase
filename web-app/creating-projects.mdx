# Using the Project Editor

The Projects space in my.tribble.ai allows users to upload and manage RFXs and questionnaires for automated responses. This guide walks you through creating and managing projects.

<Frame>
  <img src="/images/web-app/projects-dashboard.png" alt="Projects Dashboard showing active, in analysis, pending, and finished projects" />
</Frame>

## Salesforce Connection (Optional)

Before linking projects to Salesforce opportunities, ensure your Salesforce integration is set up:

1. Access Personal Settings
   * Click the gear icon in the bottom left corner of my.tribble.ai
2. Connect Salesforce Account

<Frame>
  <img src="/images/web-app/salesforce-settings.png" alt="Salesforce integration settings" />
</Frame>

## Creating a New Project

1. Access the Projects screen from the left sidebar
<Frame>
  <img src="/images/web-app/projects-menu.png" alt="Projects menu item in sidebar" />
</Frame>

2. Click "New Project"
3. Enter Project Details:
   * **Project Title** - Name your project
   * **Client Name** - Add your client's name
   * **Due Date** - Set project deadline
   * **Project Owner** - Assign project ownership
   * **Salesforce Opportunity** (optional) - Link to Salesforce
   * **Tags** - Add tags to segment content for answer generation
   * **Upload Files** - Add your RFX or questionnaire document(s)

<Frame>
  <img src="/images/web-app/new-project-form.png" alt="New project creation form" />
</Frame>

<Note>
Questionnaires with .xlsx file format must be completed one at a time and without other document attachments.
</Note>

4. Click "Continue"
   * Tribble will begin analyzing the document and generating responses

<Frame>
  <img src="/images/web-app/project-filled.gif" alt="Filled project form example" />
</Frame>
